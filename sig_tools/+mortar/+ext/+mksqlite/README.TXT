mksqlite implementiert eine Schnittstelle von MATLAB M-Skript
zur SQL Datenbank SQLite.

Installation:
Einfach den Inhalt der ZIP-Datei in einem Verzeichnis im Suchpfad von MATLAB
installieren oder ein neues Verzeichnis in den Suchpfad aufehmen.

Verwendung:
In der ZIP-Datei befindet sich ein M-Skript mit einer kurzen Befehlsreferenz.
Diese kann in MATLAB mit "help mksqlite" aufgerufen werden.
Im "doc" Verzeichnis befindet sich eine kurze Dokumentation.
Des weiteren dient das Skript "sqlite_test.m" als Beispiel für die 
Verwendung vom mksqlite.

------------------------------------------------------------------------

mksqlite implements an interface from MATLAB M-Script 
to the database Engine SQLite.

Installation:
Extract the content of the ZIP-File to a directory inside the search path of
MATLAB or use e new directory and insert it into the serach path.

Usage:
You can find a short description in the "doc" Folder.
There is an Example script inside the archive.
The MATLAB command 'help mksqlite' works - but the help page is
only german yet.
